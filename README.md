# Timelex Modern

A professional time tracking and project management application designed for freelancers, consultants, and service-based businesses.

## What It Does

Timelex Modern helps you track time spent on client projects, manage billing rates, and organize your work efficiently. It automatically calculates earnings based on tracked time and hourly rates, making billing seamless and accurate.

## Why It's Useful

- **Automated Billing**: Calculates earnings automatically from tracked time and custom hourly rates
- **Project Organization**: Keeps work organized by clients, projects, and tasks with deadlines
- **Client Management**: Maintains client profiles with individual rates and currencies
- **Time Analytics**: Provides detailed reports on time spent and earnings
- **Professional Workflow**: Supports multi-client projects with proper access control

## Key Features

- Real-time task tracking with start/stop functionality
- Multi-view dashboard (tasks, projects, clients, reports, overview)
- Project organization with color coding and deadline tracking
- Client management with hourly rate and currency support
- Task priorities, recurrence patterns, and due dates
- Token-based authentication with role-based access
- Email integration for user management

Built with Django REST API backend and React TypeScript frontend for a modern, scalable solution.
